Keyboard Shortcuts

Submission & Interrupts

EnterSend message / submit
\ + EnterNew line (without sending)
Shift+TabCycle Modes: Normal → Auto-Accept → Plan
EscInterrupt / stop generation
Ctrl+CCancel operation (hard stop)
Ctrl+DExit Claude Code
Ctrl+F UNVERIFIEDToggle Fast Mode (runs Opus at ~2.5x speed)

Navigation & UI

Ctrl+TToggle task list
Ctrl+GOpen external editor for long prompts
Ctrl+VPaste image (screenshot/diagram)
Cmd+PQuick model picker
Ctrl+BSend running task to background

CLI Launch Flags

Launch Types

claude -p "query"Print mode (headless script execution)
claude -cContinue last session
claude -w name PROStart in isolated git worktree
--from-pr <number>Resume session linked to a PR
--fork-sessionFork existing session when resuming

Behavior & Context

--permission-mode planStart in Plan Mode (read-only)
--model opusForce specific model
--max-budget-usd NSet hard spend limit
--fallback-model <name>Fallback model if primary overloaded
--json-schema '<json>'JSON schema for structured validation
--add-dir ../pathAdd extra directories to context
--remoteStart web-accessible remote session
--mcp-debugEnable MCP server debug logging

Extension System & Auth

The Big 5

1. CLAUDE.md
Project memory read every session (architecture, conventions). Create via /init.
2. Commands
Markdown prompts YOU invoke. Stored in .claude/commands/.
3. Skills
Knowledge CLAUDE auto-invokes. Stored in .claude/skills/.
4. Sub-Agents
Separate AI instances (reviewer, debugger). Manage via /agents.
5. MCP Servers
External tool connections (GitHub, DBs). Setup: claude mcp add.

Auth Commands

claude auth loginAuthenticate / re-authenticate account
claude auth statusCheck current authentication status
claude auth logoutLog out of the current session
claude updateUpdate Claude Code to latest version

Hooks & File Structure

Hooks (Event Triggers)

PreToolUseModify or block tool input before execution
PostToolUseRun linter/formatter automatically after file edit
SessionEndTrigger script when session closes
SubAgentStartFires when a sub-agent spins up
SubAgentStopFires when a sub-agent completes

File Structure Map (.claude/)

CLAUDE.mdProject memory (conventions, context)
settings.jsonShared project settings
commands/Project slash commands (*.md)
skills/Project skills (SKILL.md)
agents/Project sub-agents (*.md)
plugins/Project-level plugins directory

Workflow & Custom Commands

Most Used Combos

Plan → Execute LoopShift+Tab (Plan) → Review → Shift+Tab (Normal)
Review Git Diffgit diff | claude -p "review"
Explain Log Errorcat error.log | claude -p "explain"
Parallel Branchesclaude -w feature-a + claude -w feature-b
Save TokensRun /compact or replace pasted code with @ refs

Custom Commands Variables

$ARGUMENTS
Replaced with inputs typed after the command (e.g., /project:review src/app.ts).
$FILE_PATH UNVERIFIED
Resolves to path of edited file (for use in hook strings, not frontmatter).
